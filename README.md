# Alphacamp 2-3 Middleware 實作練習

## 功能
- 將每一比對路由的請求紀錄輸出至終端

### 使用工具
- Visual Studio Code
- Node.js
- Express

### 安裝步驟

請開啟終端機依序進行以下操作：

- git clone https://github.com/alex2116/Middleware-.git
- cd Middleware-
- npm install
- npm run dev
- 使用瀏覽器開啟：http://localhost:3000 
- 於終端機按下Ctrl/Command + C即可結束
